House Price Prediction – Model Validation, Overfitting Control & Hyperparameter Tuning

Overview

This project demonstrates advanced machine learning techniques for improving the performance and reliability of a house price prediction model. It focuses on validating model performance, detecting overfitting, and optimizing model parameters using scikit-learn.

Objectives

- Detect overfitting by comparing training and testing performance.
- Evaluate model reliability using 5-fold Cross-Validation.
- Optimize a Decision Tree Regressor using GridSearchCV.
- Compare the performance of Linear Regression, Ridge Regression, and the tuned Decision Tree model.
- Select the best-performing model based on RMSE and R² score.

Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook / Google Colab

Machine Learning Techniques

- Train-Test Split
- Decision Tree Regression
- Linear Regression
- Ridge Regression
- Cross-Validation
- Hyperparameter Tuning (GridSearchCV)
- Model Evaluation using RMSE and R² Score

Project Outcome

The project successfully validates the machine learning model, reduces overfitting through hyperparameter tuning, and identifies the most suitable regression model for accurate house price prediction.
